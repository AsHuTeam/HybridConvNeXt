# Dual-Attention Ablation Study (Multi-file GitHub Version)

This repository contains a complete ablation study for dual-attention ConvNeXt-based architectures.

## Run
1) Install requirements:
```bash
pip install -r requirements.txt
```

2) Edit dataset path in `src/config.py` (Config.data_root)

3) Run:
```bash
python main.py
```

## Outputs
Saved under:
- `ablation_results_dual_attentionnew/models/`
- `ablation_results_dual_attentionnew/plots/`
- `ablation_results_dual_attentionnew/reports/`
- `ablation_results_dual_attentionnew/ablation_comparison.csv`
- `ablation_results_dual_attentionnew/SUMMARY_REPORT.txt`
